State

public struct State

Structure that represents the store state. The store state is kept as a Dictionary with String Keys and Any Values ([String: Any])

For example, the state with two struct looks like:

[
  "TodoItems": TodoItems(....),
  "AppSettings": AppSettings(....)
]
